# URMortgage.online — SEO Content Platform

Global mortgage and property knowledge platform. 25 countries, 250+ lenders, full SEO content hubs.

## Stack
- **Framework:** Astro SSG (TypeScript)
- **Styling:** Tailwind CSS (Precision Finance design system)
- **Hosting:** Cloudflare Pages
- **Content Pipeline:** Airtable → Claude/ChatGPT → Markdown → Git → Deploy
- **Analytics:** Google Search Console + Cloudflare Web Analytics

## Project Structure
```
ai/              → System prompt, architecture rules, coding standards
tasks/           → Backlog, current sprint, lessons, reviews
agents/          → Planner, architect, coder, tester role definitions
docs/            → Technical architecture
src/             → Application code (Astro project — Phase 2)
```
